# Todo App

A simple responsive Todo Web application built using only HTML, CSS, and Vanilla JS.

## Features Implemented

- Add new todo tasks
- Display all todos in a list
- Mark todos as completed
- Completed todos show strikethrough text
- Delete todos
- Input validation to prevent empty tasks
- Responsive design for desktop and mobile
- Edit existing tasks
- Filter todos by All, Active, and Completed
- Show total and completed task counts
- Store todos in Local Storage, so data remains after page refresh
Basicely all requriment completed according to the assignment.

## How to Run

1. Download or clone this project.
2. Open the folder.
3. Open `index.html` in any modern browser like Firefox or Chrom.

## Assumptions and Limitations

- Todos are stored only in the browser's Local Storage.
- Data is available only on the same browser and device.
- No backend or database is used.
